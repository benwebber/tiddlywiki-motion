# tiddlywiki-plugins

A collection of useful plugins for [TiddlyWiki](https://tiddlywiki.com/).

## Plugins

<dl>
  <dt>[expand][]</dt>
  <dd>Expand upon terms without breaking the flow of text</dd>
  <dt>[motion][]</dt>
  <dd>Vim-like keyboard shortcuts</dd>
</dl>
